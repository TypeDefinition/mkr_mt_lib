Subscribes a listener to this dispatcher for this type of event. A listener can subscribe multiple times. A listener can subscribe to multiple events from the same dispatcher. A listener can subscribe to multiple events from different dispatchers. A listener can subscribe to the same event from different dispatchers. A listener can subscribe to the same event from the same dispatcher multiple times.
